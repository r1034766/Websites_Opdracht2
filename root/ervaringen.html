Enkele belangrijke certificaten en opleidingen.

TIA Portal

De dag van vandaag is TIA Portal één van de meest gebruikte tools voor het automatiseren in de industriële sector. Deze tool is ontwikkeld door Siemens en wordt onder andere gebruikt om de typische Siemens PLC te programmeren. Deze tool heeft uiteraard nog veel meer functionaliteiten wat het gebuik ervan ook zeer complex maakt. Ik heb via mijn wergever de kans gekregen om de TIA Portal opleiding te volgen. Voor deze tool heb ik de basis- en geavanceerde opleiding gevolgd.

BA5

Werken aan industriële elektrische installaties kan zeer gevaarlijk zijn. Om het risico op ongevallen te beperken is het daarom zeer belangrijk om goed geïnformeerd te zijn. Alvorens het BA5 certificaat wordt gegeven, moet er een test worden afgelegd om te bewijzen dat de kandidaat de belangrijkste principes vanuit de elektriciteit onder de knie heeft. Via dit certificaat weet de werkgever dat de technieker de juiste competenties heeft om veilig aan elektrische installaties te werken. Zonder dit certificaat is het verboden om zelfstandig aan een elektrische installatie te werken. In 2020 heb ik dit certificaat behaald.

Aan welke projecten heb ik gewerkt?

Klimaatkasten KBC Mechelen

Bij dit project werden een tiental klimaatkasten voor een groote serverroom vervangen door nieuwe. Ook de bestaande sturing van de HVAC installatie moest een upgrade krijgen.

Mijn taak bestond erin om de volledige sturing van de HVAC en de klimaatkasten te vernieuwen. Aangezien alles nog via verouderde technologie gestuurd werd, moest ik voor de migratie naar de nieuwe controllers volledig van nul beginnen. Dit was zeker niet vanzelfsprekend aangezien de programmatie van de oude technologie gebaseerd was op code en de nieuwe controllers via functieblokken moesten worden geprogrammeerd.

De integratie van de klimaatkasten op het bestaande gebouw beheer systeem was ook een uitdaging in de zin dat deze niet hetzelfde communicatie protocol gebruikten als het bestaande Siemens materiaal. Om de klimaatkasten toch te kunnen laten communiceren met het bestaande systeem werd er gebruik gemaakt van een integratie via "Modbus". Dit is een standaard communicatie protocol dat compatibel is met een Siemens PLC mits een bijkomende module wordt gebruikt in combinatie met de PLC.

Aangezien ik voor dit project nooit in aanraking ben gekomen met Modbus integraties, heb ik nog snel een on-the-job training gevolgd om zo de nodige kennis op te doen om de verdere indienststelling van de klimaatkasten te kunnen afwerken. Dit is uiteindelijk ook gelukt waardoor ik de klant tevreden heb kunnen achterlaten met een werkende serverroom.


Labo's OCAS

Voor dit project was ik verantwoordelijk voor de programmatie van een nieuw laboratorium bij OCAS in Zelzate. OCAS (OnderzoeksCentrum voor de Aanwending van Staal) is een geavanceerd, marktgericht onderzoekscentrum dat wereldwijd staal- en metaalgebaseerde producten, diensten en oplossingen levert aan metaalverwerkende bedrijven.

Dit project was iets complexer omdat het een laboratorium betrof, waarbij speciale aandacht moest worden besteed aan de boven- en onderdruk in de laboratoria. Bij andere projecten was dit minder relevant. Bovendien moest de nieuwe PLC worden geïntegreerd in het bestaande, oudere systeem, wat enkele extra tussenstappen vereiste. Tot slot was het mijn taak om de nieuwe laboratoria te visualiseren in het bestaande gebouwbeheersysteem.
